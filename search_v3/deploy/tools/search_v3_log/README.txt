一、log_manager.py:
说明：搜索日志管理程序;需要util.py辅助程序（所需功能：读取配置文件的搜索日志所在的目录）
功能：删除当前目录下的搜索日志
参数说明：
 1. -t/--time  日期时间（eg:2012-06-06）#删除日期时间前的所有搜索日志
 2. -d/--day   整型天数(eg:3) #删除天数前的所有搜索日志
 3. -w/--week  整型周数(eg:3) #删除周数前的所有搜索日志
 4. -f/--file  配置文件（存储日志文件的目录）
 5. 若无参数输入，默认为当天,默认配置文件为“./log.conf”。

二、syn_time.py
说明：同步时间程序;需要util.py辅助程序（所需功能：读取配置文件的数据库连接信息）
功能：同步搜索后台和数据库的时间（以数据库时间为准）
参数说明：
 1. -f/--file     配置文件，可参考db.conf文件
 2. 若无参数输入，默认配置文件为“./db.conf”
